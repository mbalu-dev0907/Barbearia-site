# 💈 Barbearia Elite

Um site simples e moderno para uma barbearia, desenvolvido como projeto inicial de portfólio.

---

## 🚀 Demonstração
Este projeto simula o site de uma barbearia com apresentação de serviços e contato.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📂 Estrutura do projeto

/barbearia-site
│── index.html
│── style.css
│── script.js

---

## ✨ Funcionalidades

- Página inicial com apresentação
- Lista de serviços
- Seção de contato
- Botão interativo com JavaScript

---

## 🎯 Objetivo

Este projeto foi criado para praticar desenvolvimento web e servir como base para projetos profissionais.
